# Aayush Portfolio

Static portfolio built with HTML, CSS, and React via CDN.

## Files
- index.html
- styles.css
- projects.js
- app.jsx
- assets/aayush-photo.jpeg
- assets/Aayush_Sapkota_CV.pdf

## Deploy on GitHub Pages
1. Upload the contents of this folder to the root of your public repository.
2. Ensure `index.html` is in the root.
3. In GitHub, go to Settings > Pages.
4. Choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save and wait for the site to publish.
